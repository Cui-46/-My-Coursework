## 基本信息

课程名称：三维动画基础

## 项目简介

这个作业制作了一段纯室外哥特悬疑风格的 3D 短片，围绕“侦探深夜调查废弃吸血鬼城堡，并遭遇吸血鬼红光攻击”的故事展开。最终成果是一段包含城堡外景、墓地环境、侦探探索、吸血鬼显现、红色能量攻击和悬念结尾的短片。

## 制作过程

1. 前期调研：确定短片主题为纯室外哥特悬疑场景，参考吸血鬼城堡、墓地、侦探等元素，明确黑暗神秘的整体风格。
2. 方案设计：设计“侦探深夜来到废弃城堡外调查，发现吸血鬼”的故事流程，并规划城堡外景、墓地、侦探、吸血鬼和手电筒等主要内容。
3. 实现或制作：使用 AI 生成概念图和 3D 模型，将模型导入 Blender，搭建城堡墓地场景，制作侦探调查、手电筒扫光和吸血鬼出现等动画。
4. 测试与修改：通过预览渲染检查场景布局、灯光效果、角色动作和镜头节奏，并进行相应调整。
5. 最终展示：最终完成一段纯室外哥特悬疑 3D 短片，呈现侦探在月夜城堡外调查并遭遇吸血鬼的过程。

## 总结反思

学习收获：通过这个作业，我了解了 AI 生成图片、图生 3D 模型和借助AI工具如Claude Code制作 Blender 动画制作的基本流程，也学习了如何搭建室外场景、安排镜头、设置灯光和营造悬疑氛围。
遇到的问题：制作过程中遇到了镜头不够合理，模型比例不统一、场景摆放不够自然、灯光太暗或太亮、角色动作不够流畅等问题，需要不断预览和调整。
后续改进：之后可以进一步优化角色动作和镜头衔接，增加更细致的环境细节，例如雾气、树影、风吹斗篷等效果，让短片画面更真实，氛围更强。
